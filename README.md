# 🏷️ Tag Creator — Interactive Tag Input Component

A lightweight, **zero-dependency** tag input component built with pure HTML, CSS, and Vanilla JavaScript. No frameworks, no libraries — just clean, reusable web code.

---

## ✨ Features

- ⌨️ **Full Keyboard Navigation** — Add tags with `Enter` or `,` and remove with `Backspace`
- 🎨 **Animated Label Design** — Smooth floating label animation on focus
- 💫 **CSS Transitions** — Fluid animations for tag creation and removal
- 🔌 **Zero Dependencies** — No npm, no frameworks, no external libraries
- 📦 **Easily Embeddable** — Drop it into any web project with minimal setup
- 📱 **Fully Responsive** — Works across all screen sizes

---

## 🎮 How to Use

### Add a tag
- Type a word and press `Enter` or `,`

### Remove a tag
- Click the `×` on a tag, or press `Backspace` to remove the last one

### Embed in your project
```html
<!-- 1. Add the HTML structure -->
<div class="tag-input-wrapper">
  <input type="text" id="tag-input" placeholder="Add a tag..." />
  <div id="tags-container"></div>
</div>

<!-- 2. Link the CSS -->
<link rel="stylesheet" href="style.css" />

<!-- 3. Link the JS -->
<script src="script.js"></script>
```

---

## 💡 Why This Project?

Most tag input solutions require heavy libraries like React Select or jQuery plugins. This component proves that a **polished, production-ready UI** can be achieved with just the browser's native APIs — resulting in faster load times and maximum compatibility.
